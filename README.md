# Bloom — Learn English

A fully offline, installable PWA (Progressive Web App) for practising English.
No internet connection required after installation. No API keys. No server. No cost.

## What's inside

- **Placement test** — 8 questions that recommend a starting CEFR level (A1 to C2)
- **Vocabulary flashcards** — 6 words per level, with IPA, definition, and example sentence
- **Grammar quiz** — 5 questions per level with a rule reminder and instant feedback
- **Reading practice** — a level-appropriate passage; tap highlighted words for definitions
- **Listening comprehension** — read a dialogue and answer questions (simulated listening)
- **Speaking practice** — guided prompts with model answers to compare your own response
- **Level progression** — every activity earns points; 500 points advances you to the next level
- **Daily streak** — opens every day to keep a streak counter going
- **Progress saved** — level, points, words learned, and streak are stored in your browser

## Project structure

```
english-pwa/
├── public/          Icons, favicon, PWA manifest (auto-generated by Vite plugin)
├── src/
│   ├── components/  All screens: Home, Vocabulary, Grammar, Reading,

│   │                levels.js · vocabulary.js · grammar.js

├── vite.config.js   Vite + PWA plugin (manifest, service worker, offline cache)
└── package.json
```

## Running locally

You need [Node.js](https://nodejs.org) 18 or newer.

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Building for production

```bash
npm run build
```

This creates a `dist/` folder — deploy it to any static host (Netlify, Vercel,
GitHub Pages, Cloudflare Pages). No server or backend is required at all.

## Installing as an app

Once the `dist/` folder is deployed and opened in a mobile or desktop browser:

- **iPhone (Safari)** → Share → "Add to Home Screen"
- **Android (Chrome)** → menu → "Install app"
- **Desktop (Chrome/Edge)** → click the install icon in the address bar

The app opens full-screen with its own icon, works completely offline, and
remembers your progress between sessions.

## Customising the content

All lesson content lives in `src/data/`. Each file exports an object keyed by
CEFR level (`A1` through `C2`). To add new words, grammar topics, reading
passages, listening dialogues, or speaking prompts — just follow the existing
shape in the relevant file and they will appear immediately at the right level.

No other code needs to change.
